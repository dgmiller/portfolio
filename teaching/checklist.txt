TO BE USED AS A TEMPLATE FOR EACH PROJECT

=========================================

I. Define the Problem

	> What is the problem?
	> Why does the problem need to be solved?
	> How would I solve the problem?

II. Prepare Data

	> Data selection
	> Data preprocessing
	> Data transformation

III. Spot-check Algorithms

	> Test all major algorithm families
	> Find those that succeed in picking out the structure of the problem
	> Perform more focused experiments

IV. Improve Results

	> Algorithm Tuning
	> Ensemble Methods
	> Extreme Feature Engineering

V. Present Results

	> Context (Why)
	> Problem (Question)
	> Solution (Answer)
	> Findings
	> Limitations
	> Conclusions (Why + Question + Answer)

=========================================

ALGORITHMS

I. Regression

	> Linear Regression (OLS)
	> Logistic Regression
	> Stepwise Regression
	> Multivariate Adaptive Regression Splines (MARS)
	> Locally Estimated Scatterplot Smoothing (LOESS)

II. Instance-based

	> k-Nearest Neighbor (kNN)
	> Learning Vector Quantization (LVQ)
	> Self-Organizing Map (SOM)
	> Locally Weighted Learning (LWL)

III. Regularization

	> Ridge Regression
	> Least Absolute Shrinkage and Selection Operator (LASSO)
	> Elastic Net
	> Least-Angle Regression (LARS)

IV. Decision Tree

	> Classification and Regression Tree (CART)
	> Iterative Dichotomiser 3 (ID3)
	> C4.5 and C5.0
	> Chi-squared Automatic Interaction Detection (CHAID)
	> Decision Stump
	> M5
	> Conditional Decision Trees

V. Bayesian

	> Naive Bayes
	> Gaussian Naive Bayes
	> Multinomial Naive Bayes
	> Averaged One-Dependence Estimators (AODE)
	> Bayesian Belief Network (BBN)
	> Bayesian Network (BN)

VI. Clustering

	> k-Means
	> k-Medians
	> Expectation Maximization (EM)
	> Hierarchical Clustering

VII. Association Rule Learning

	> Apriori
	> Eclat

VIII. Artificial Neural Network

	> Preceptron
	> Back-propagation
	> Hopfield Network
	> Radial Basis Function Netowrk (RBFN)

IX. Deep Learning

	> Deep Boltzmann Machine (DBM)
	> Deep Belief Networks (DBN)
	> Convolutional Neural Network (CNN)
	> Stacked Auto-Encoders

X. Dimensionality Reduction

	> Principal Component Analysis (PCA)
	> Principal Component Regression (PCR)
	> Partial Least Squares Regression (PLSR)
	> Sammon Mapping
	> Multidimensional Scaling (MDS)
	> Projection Pursuit
	> Linear Discriminant Analysis (LDA)
	> Mixture Discriminant Analysis (MDA)
	> Quadratic Discriminant Analysis (QDA)
	> Flexible Discriminant Analysis (FDA)

XI. Ensemble

	> Boosting
	> Bootstrapped Aggregation (Bagging)
	> AdaBoost
	> Stacked Generalization (blending)
	> Gradient Boosting Machines (GBM)
	> Gradient Boosted Regression Trees (GBRT)
	> Random Forest
